Welcome to the zyBooks zip file extractor and file renamer.

This app was built out of frustration with the download format of the zyBooks assignment downloads.

Usage:

Select an assignment folder using the folder icon in the main window. The zipRenamer app will examine ALL subfolders for zip files.

It assumes that there is a single Python file in each zip file, which it will rename to use either the student's name, the student'n name and email or the entire filename string. The choice of these is made in the preferences dialog accessed via the gear icon on the main screen.

You can also choose to remove the zip files when you are done using this dialog screen.
